# 🧨 VirusPack‑c00lkiddEdition
Colección experimental creada por Anyelo (c00lkidd) que reúne ejecutables, scripts, mensajes ocultos y archivos con lore digital.
Este pack mezcla caos controlado, humor, estética hacker y mini‑ARGs diseñados para sorprender, experimentar y jugar con la curiosidad del usuario.

## 📁 Contenido del repositorio
end.exe
Simula el cierre definitivo del pack.
Actúa como “final boss” del VirusPack.

## mensaje_final.txt
Documento que revela el mensaje oculto del pack.
Sirve como cierre narrativo del mini‑ARG.

## virus.exe / virus2.exe / virus3.exe
Ejecutables experimentales que forman la secuencia principal del pack.
No son virus reales.
Incluyen:

mensajes ocultos

animaciones

efectos visuales

simulaciones de error

lore progresivo

Cada uno representa una “fase” del caos.

## 🎮 Propósito del proyecto
VirusPack‑c00lkiddEdition existe para:

documentar los experimentos de Anyelo

preservar el lore de los archivos

mostrar cómo funcionan los .exe y .bat

servir como base para futuros packs

permitir que otros exploren el caos creativo del autor

## ⚠️ Advertencia
Ningún archivo es dañino.
Todo es experimental, educativo y parte del estilo creativo del autor.
El pack está diseñado para sorprender, no para afectar sistemas reales.

## 🧩 Lore del VirusPack
La secuencia del pack sigue esta narrativa:

virus.exe — Inicio del caos

virus2.exe — Escalamiento del evento

virus3.exe — Revelación del mensaje oculto

mensaje_final.txt — Explicación del propósito

end.exe — Cierre definitivo del pack

Cada archivo forma parte de un mini‑ARG estilo c00lkidd.
